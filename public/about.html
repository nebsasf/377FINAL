About

SkillServe makes volunteering easier to discover.

Many people want to help their communities but do not know where to start. SkillServe organizes volunteer opportunities by category, location, and community need so users can quickly find a place to contribute.

The application uses an Express backend, Supabase database, Fetch API calls, Chart.js visualizations, Leaflet maps, and the Open-Meteo external API.
